# Universal AI Engineering Protocol
Source of truth for cross-device development standards, token optimization, and AI collaboration rules.

## Quick Start (New Machine)
Run this command in PowerShell to sync your environment:
```powershell
powershell -ExecutionPolicy Bypass -File scripts/bootstrap.ps1
```

## Structure
- `/knowledge`: Core principles, token strategies, and Karpathy's coding skills.
- `/scripts`: Automation and setup scripts.
- `/configs`: (Future) MCP server configurations and IDE settings.
